Conway's Game of Life, mashallah ++ premium deluxe GOTY édition
Cliquez sur une cellule morte pour lui donner vie, et sur une cellule vivante pour la tuer. Appuyez sur start pour lancer la simulation.

Les règles

  • Si une cellule vivante a moins de 2 voisins vivants, elle meurt de souspopulation
  • Si une cellule vivante a 2 ou 3 voisins vivants elle reste vivante
  • Si une cellule vivante a plus de 3 voisins vivants, elle meurt de surpopulation
  • Si une celulle morte a 3 voisins vivants, elle devient vivante par reproduction
